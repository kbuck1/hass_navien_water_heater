# Navien Water Heater Custom Integration
Control and monitor the state of Navien water heaters which are connected to the Navien cloud portal via a [NaviLink or NaviLink Lite](https://www.navieninc.com/accessories/navilink) control system.

> ⚠️ **WARNING: This code is in-development and may not be stable.**
> 
> This is a development fork of the Navien Water Heater integration. For the stable, production-ready version, please use the [original repository](https://github.com/nikshriv/hass_navien_water_heater) maintained by nikshriv.

This fork attempts to implement NWP500 support.
